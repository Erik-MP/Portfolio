Algunos de mis proyectos son:

Pagina de Joyeria (TFG)

Este proyecto consistió en hacer una pagina web en la que se pudiera registrar un usuario, iniciar sesion, realizar una compra. Los datos de la pagina web estan conectados a una base de datos que guarda todos los productos, usuarios y pedidos que se hacen en ella.

Pagina de Camisetas

Este proyecto consistió en hacer una pagina web para una tienda online de personalizacion de camisetas. Hice 2 sketches a lapiz y papel y elegir uno de los diseños, realizar el wireframe del diseño y realizar un prototipo de la pagina.
